# Incursa.Quic

[`Incursa.Quic`](../../README.md) is the packable library root for the Incursa QUIC work. The project is intentionally empty for now and exists to hold the build, packaging, and documentation baseline.

## Install

```bash
dotnet add package Incursa.Quic
```

## Status

- No runtime types have been added yet.
- Package metadata, versioning, and packaging are already configured in the repository root.
- Future QUIC implementation code can be added here without changing the surrounding repo structure.
