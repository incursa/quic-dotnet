# Specs

This tree contains the canonical artifact families used by the repository. For the SpecTrace document families, sibling `.json` files are canonical.

## Canonical Areas

- [`requirements/`](requirements/README.md): specifications and requirement gaps.
- [`architecture/`](architecture/README.md): design artifacts that satisfy requirements.
- [`schemas/`](schemas/README.md): canonical SpecTrace JSON schemas.
- [`work-items/`](work-items/README.md): implementation work that addresses requirements.
- [`verification/`](verification/README.md): proof artifacts for requirement sets.
- [`generated/`](generated/README.md): derived traceability outputs and review material.
- [`templates/`](templates/README.md): copy-ready canonical JSON templates.
