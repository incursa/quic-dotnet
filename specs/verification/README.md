# Verification

This directory holds verification artifacts for canonical requirement sets.
Canonical verification artifacts are authored in `.json`.

## Domain Areas

- [`quic/`](quic/README.md): QUIC verification artifacts

Verification files should state what was proved, how it was proved, and which requirement IDs were covered.
